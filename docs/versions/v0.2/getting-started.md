# Getting Started with CapyKit v0.2.0

This guide will help you get started with CapyKit v0.2.0.

## Installation

```bash
npm install capykit@0.2.0
```

## Quick Start

```vue
<template>
  <div>
    <CapyButton>Click me</CapyButton>
  </div>
</template>

<script setup>
import { CapyButton } from 'capykit'
</script>
```
