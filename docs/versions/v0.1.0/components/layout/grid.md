# Grid Component

A responsive grid system.

## Usage

```vue
<template>
  <CapyGrid cols="3" gap="4">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
  </CapyGrid>
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| cols | number | 1 | Number of columns |
| gap | number | 2 | Gap between items |
